Inilah materi konvensional KURSUS WEBSITE Pada Umumnya

Jika anda ingin menjadi seorang Web Developer yang bisa membangun website ideal setidaknya anda harus paham beberapa istilah teknis coding dan server seperti berikut :

HTML

Merupakan bahasa pemrograman standar yang membuat halaman website bisa tampil di browser seperti chrome & mozilla

CSS

Bahasa pemrograman yang membuat tampilan website menjadi indah berwarna warni dan tata letak yang rapi

PHP

Bahasa pemrograman yang dieksekusi oleh komputer server sehingga menghasilkan halaman HTML.

APACHE/NGINX

Merupakan web server yang berjalan di komputer server sehingga website dapat tampil ketika ditulis alamatnya di browser

MYSQL

Merupakan aplikasi database penyimpan data yang berjalan di komputer server. Semua bentuk teks disimpan di aplikasi ini

LINUX

Sistem Operasi (seperti windows 10 untuk laptop) yang umum digunakan pada kebanyakan komputer server.

Perlu waktu berbulan-bulan agar seseorang bisa menguasai semua skill tersebut. Hanya programmer sejati yang antusias mempelajari semua itu

Wordpress, CMS Populer Sejuta Umat

Untuk melompat jauh ke depan, biasanya orang akan meyarankan WORDPRESS untuk membuat website dalam bentuk apa saja. Selain itu, mayoritas orang hanya memanfaatkan sebagian kecil dari kekuatan wordpress yang dibangun dalam ribuan baris kode ini. ADA CARA YANG LEBIH CEPAT

Apa itu Wordpress?

Wordpress adalah file website sudah jadi dan siap pakai. Terdiri dari ribuan file yang dibuat oleh komunitas programmer handal dari seluruh dunia. Wordpress dapat didownload siapapun dan diinstall di server. Berdasarkan data saat ini, 30% lebih website di seluruh dunia menggunakan WORDPRESS.

Berapa Biaya Pembuatan Website Menggunakan Wordpress?

Wordpress memang bisa diperoleh secara gratis. Namun Jika anda ingin membuat website menggunakan wordpresss, anda harus menyiapkan domain dan server hosting dengan biaya tahunan antara 400 ribu hingga jutaan rupiah.

Selain itu, anda juga harus menyesuaikan beberapa pengaturan (Tweaking) dan tampilan agar website menjadi sesuai idaman anda : Cepat, Cantik, dan Mudah ditemukan saat anda googling. Yang ternyata sedikit sangat rumit untuk anda yang tidak memiliki basic IT
